Extensions

Les extensions

Pour étendre les fonctionnalités du CMS WordPress, il est possible d’installer des extensions.

Il existe plusieurs dizaines de milliers d'extensions mais il est important de ne pas surcharger votre site internet avec trop d'extensions !

  • Augmentation des failles de sécurité
  • Interface d’administration compliquée
  • Ralentissement du temps de chargement des pages web
  • Néfaste pour le SEO
  • Risque de conflits entre les extensions
N'installez que les extensions nécessaires !

JetPack

Jetpack est un plugin développé pour le site internet wordpress.com

Plusieurs fonctionnalités sont disponibles gratuitement, il faut ensuite payer pour accéder à plus de fonctionnalités :

  • Newsletter
  • Formulaire de contact
  • SEO
  • Sauvegarde
  • Performance
  • ...

SEO

Plusieurs extensions peuvent être installés afin d’optimiser le référencement.

Attention cependant à ne pas installer 2 extensions réalisant la même tâche et qui pourraient alors entrer en conflit !

  • All In One SEO
  • Yoast SEO
  • Google XML Sitemaps

Formulaires

Afin de permettre aux internautes de vous contacter, vous pouvez également installer une extensions qui vous permettra de générer un formulaire de contact.

  • Contact Form 7
  • WP Forms
  • Ninja Forms

Réseaux sociaux

Dans le but d’améliorer la visibilité de votre site internet, vous pouvez installer des extensions pour lier votre site aux réseaux sociaux.

  • AddToAny Share Buttons
  • Social Media Share Buttons & Social Sharing Icons
  • Instagram Feed
  • Custom Facebook Feed

Newsletter

Une extensions de newsletter vous permettra d’envoyer régulièrement des mails d’informations à vos internautes.

  • Newsletter
  • MailChimp for WordPress
  • MailPoet

Slider

WordPress dispose d’un système de galerie assez rudimentaire. Il est donc possible d’installer une extension pour faciliter l’intégration d’un slider.

Cependant, il est parfois plus simple d’intégrer à la main en HTML/JS un slider disponible sur internet.

  • MetaSlider
  • SlideAnything
  • WP Slick Slider and Image Carousel

Shortcodes

Les shortcodes permettent d’insérer du contenu dynamique dans un article.

Exemple : [gallery size="medium"]

  • Jetpack
  • Shortcodes Ultimate

Sécurité

Les sites WordPress étant très souvent victimes d’attaques, il est fortement recommandé d’installer des extensions complémentaires pour améliorer la sécurité d’un site internet :

  • Akismet Anti-Spam
  • Limit login attempts
  • Really Simple SSL
  • Really Simple CAPTCHA
  • Wordfence Security
  • iThemes Security

Performances

Si votre site internet contient beaucoup de contenus ou si vous avez de nombreux internautes visitant vos pages web, il peut être intéressant d’installer des extensions pour optimiser les performances :

  • WP super cache (pages HTML statiques)
  • WP Fastest Cache
  • Smush Image Compression and Optimization

Ces extensions vous permettent d’éviter de regénérer le contenu d’une page à chaque chargement, de limiter les accès à la base de données, d’optimiser les images, de minifier et regrouper les fichiers CSS/JS, d’utiliser des CDN...

Sauvegarde

Afin d’éviter de perdre des données, il est primordiale d’effectuer des sauvegardes régulièrement.

Ne sauvegardez pas vos fichiers de backup directement sur le serveur (pas de sauvegarde en cas de crash du serveur !). Certaines extensions proposent des sauvegardes sur des services cloud (Google Drive, Dropbox…)

  • Duplicator
  • UpdraftPlus
  • BackWPup
  • All-in-One WP Migration

Réglementation

Afin de respecter la réglementation en vigueur, vous pouvez installer des extensions pour informer vos internautes sur l’utilisation de cookies :

  • Cookie Notice for GDPR
  • Cookie Consent

Page builder

Les page builder sont des extensions vous permettant de faciliter la création et le positionnement d’éléments dans vos pages web.

Ils proposent plusieurs templates et permettent de construire des pages web responsives. Cependant, ils rendent l’administration d’un site internet plus complexe et ne conviennent pas aux utilisateurs novices.

  • Page Builder
  • Elementor
  • Divi
  • Beaver Builder

Page builder

Avec l’arrivée du nouveau WYSIWYG Gutenberg, il pourrait ne plus être nécessaire d’installer un plugin de page builder.

Développeur

Pour les développeurs, il existe également de nombreuses extensions permettant de simplifier le développement de site internet avec le CMS WordPress :

  • ACF
  • Custom Post Type UI
  • MetaBox
  • Regenerate thumbnails

Ecommerce

Le plugin le plus largement utilisé pour concevoir un site e-commerce est WooCommerce :

  • WooCommerce
